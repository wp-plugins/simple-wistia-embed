The simplest Wistia OEmbed plugin ever. Simply paste the URL and it works!

== Description ==

This plugin allows you to use the simplest Wistia embedding workflow possible. Simply take the URL of the video you wish to embed, and paste it into your post or page. It just works!

== Installation ==

1. Upload `simple-wistia-embed.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start embedding Wistia videos!

== Frequently Asked Questions ==

= Does this work for any other video services? =

No, just Wistia.

= Does this work for custom Wistia URLs? =

Not currently, but that functionality is forthcoming.
